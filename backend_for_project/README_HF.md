# AttendVision Face Recognition Backend

Face recognition attendance system with anti-spoofing using ArcFace and MiniFASNet.

## Features
- Face detection (Haar Cascade)
- Face recognition (ArcFace via DeepFace)
- Liveness detection (MiniFASNet)
- PostgreSQL with pgvector
