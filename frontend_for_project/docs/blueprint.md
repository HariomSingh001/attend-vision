# **App Name**: AttendVision

## Core Features:

- Secure User Login: Implement a secure authentication system for teachers to log in.
- Real-time Face Recognition: Integrate a face recognition tool via webcam to automatically mark attendance when faces are detected.
- Attendance Marking: Interface to manually mark attendance with the list of detected faces or add student who could not be detected automatically.
- Attendance Dashboard: Provide at-a-glance statistics on class strength, teacher information.
- Generate Attendance Reports: Enable teachers to generate and download attendance reports.

## Style Guidelines:

- Primary color: Forest green (#34A04A) for a professional and trustworthy feel.
- Background color: Light gray (#F0F0F0) to provide a neutral and clean backdrop.
- Accent color: Steel blue (#468B97) to complement the primary green, while distinguishing important elements.
- Body and headline font: 'Inter', a sans-serif font, for a clean and modern look. Its neutrality and legibility are well-suited to an attendance system.
- Use outline-style icons to represent different attendance-related actions, such as marking present, absent, or generating reports. Use green as the primary color for the icons.
- Use a tile-based layout for the dashboard, presenting information in a clear and organized manner.
- Implement subtle animations when faces are recognized, attendance is marked, or reports are generated to provide visual feedback to the user.
